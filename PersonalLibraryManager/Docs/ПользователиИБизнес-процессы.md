# 3. Пользователи и бизнес-процессы (Users and Business Processes)

## Целевая аудитория:
Приложение предназначено для одного пользователя — владельца домашней библиотеки.

## Основные сценарии (Бизнес-процессы):

1. **Учет библиотеки:** Пользователь запускает приложение. Если база данных отсутствует, она создается автоматически.
2. **Добавление новой книги:** Пользователь открывает форму добавления, вводит название, выбирает автора (или создает нового) и жанр. Данные сохраняются в БД.
3. **Просмотр списка:** На главном экране отображается таблица всех книг.
4. **Редактирование/Удаление:** Пользователь выбирает книгу из списка и может изменить её параметры или удалить запись из библиотеки.

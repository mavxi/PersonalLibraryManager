# 8. Порядок разработки (Development Order)
Разработка проекта велась поэтапно:

1. Подготовка: Настройка репозитория, установка пакетов NuGet (Entity Framework Core, SQLite).
2. Проектирование: Создание UML-диаграмм (прецеденты, классы).
3. Создание каркаса: Реализация базового класса ViewModelBase и RelayCommand.
4. Разработка модели (Model): Создание классов Book, Author, Genre.
5. Настройка БД: Реализация LibraryDbContext, подключение SQLite, настройка строки подключения.
6. Разработка Views и ViewModels:
    * Создание MainWindow и MainViewModel.
    * Реализация AddBookWindow и логики сохранения данных.
7. CRUD операции: Реализация методов Add, Update, Delete для сущностей.
8. Отладка: Проверка корректности сохранения данных при закрытии приложения.
9. Финал: Добавление окна "О программе", рефакторинг кода, оформление документации.

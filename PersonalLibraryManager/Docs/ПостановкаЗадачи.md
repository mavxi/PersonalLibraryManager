# 1. Постановка задачи (Problem Statement)

## Целью данной учебной практики является разработка приложения "Personal Library Manager" для управления домашней библиотекой.

Приложение должно позволять пользователю:

* Хранить информацию о книгах, авторах и жанрах.
* Добавлять, редактировать и удалять записи о книгах.
* Просматривать список книг с фильтрацией или поиском (если предусмотрено).
* Автоматически создавать локальную базу данных при первом запуске.
